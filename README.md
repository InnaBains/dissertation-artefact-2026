# Sustainable Nutrition Recommendation System – Dissertation Artefact

This repository hosts the **public GitHub Pages site** for my BSc (Hons) Computing dissertation artefact:

> **Sustainable Nutrition Recommendation System: Smart Meal Planning for a Better Future**

The artefact is a **transparent, rule‑based web application** that helps UK university students generate weekly meal plans that balance:

- **Nutritional quality** (aligned with NHS Eatwell guidelines)  
- **Environmental sustainability** (CO₂e indicators from Poore & Nemecek, 2018)  
- **Household ingredient reuse** (pantry‑aware, expiry‑date‑sensitive logic)  

The full source code is stored in a **private repository** and can be made available to examiners on request.
