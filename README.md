# Sustainable Nutrition Recommendation System – Dissertation Artefact

This repository hosts the public artefact website developed for my BSc (Hons) Computing dissertation at Arden University.

## Project Title

**Design and Implementation of a Web-Based AI Framework for Sustainable Nutrition Management**

Project Overview

The Sustainable Nutrition Recommendation System is a transparent, rule-based web application designed to support healthier and more sustainable meal-planning behaviour for UK university students.

The system integrates:

Nutritional guidance and nutrition-aware recommendation logic
Environmental sustainability indicators based on CO₂e impact data
Pantry-aware recommendation behaviour
Expiry-date prioritisation and food-waste reduction strategies
Weekly meal planning
Pantry management
Explainable rule-based recommendation logic

The artefact demonstrates how nutrition, sustainability and pantry-aware food-waste reduction can be integrated within a single web-based decision-support framework.

Live Demo Notice

The application is hosted using free cloud services. If the backend has been inactive, the first request may take approximately 30–60 seconds while the service starts.

Demonstration Database Notice

The deployment uses a shared demonstration database. Pantry quantities, recommendation outputs, meal plans and planning statistics may vary as the system is tested by different users.
